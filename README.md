
# 🎁 Gift Card Game (Java)

A multi-file Java program simulating a randomized gift card drawing game. Developed as part of Software Engineering coursework to demonstrate object-oriented programming, file input, and randomized logic.

## 🔧 Features
- Custom object classes: `GiftCard`, `Barrel`, and `Main`
- Reads card data from a `.txt` file
- Simulates drawing cards from a randomized barrel
- Handles user interaction through a console interface

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Java Collections and ArrayLists
- File I/O and exception handling
- Randomization and loop control
